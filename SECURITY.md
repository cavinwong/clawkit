# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | ✅ Active support  |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in ClawKit, please report it responsibly by emailing **clawkit@openclaw.dev** with the subject line `[SECURITY] <brief description>`.

Please include the following in your report:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any suggested mitigations or fixes

You can expect an acknowledgment within **48 hours** and a resolution timeline within **7 days** for critical issues.

We appreciate responsible disclosure and will credit reporters in the release notes (unless you prefer to remain anonymous).
